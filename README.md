# TETFund Inspection Report Form

A Streamlit web application for TETFund Monitoring and Evaluation Department to submit inspection reports.

## Features
- Institution Information
- Project Inspection Details
- Monitoring Team Management
- Summary Calculations
- Print pdf of the report


## Live Demo
[View Live App](YOUR_STREAMLIT_CLOUD_URL)

## Local Development
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`

## Deployment
Deployed on Streamlit Cloud
